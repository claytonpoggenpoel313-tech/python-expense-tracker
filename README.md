# 💰 Python Expense Tracker

A simple command-line expense tracker built with Python to manage income and expenses, track transactions, and calculate balance.

---

## 🚀 Features
- Add income and expenses
- Store transactions in a JSON file
- View full transaction history
- Calculate total balance
- Simple and clean CLI interface

---

## 🛠️ Technologies Used
- Python 3
- JSON (for data storage)
- datetime module
- File handling
